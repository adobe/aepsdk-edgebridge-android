### extensionVersion
Returns the version of the Edge Bridge extension.

#### Java

##### Syntax
```java
public static String extensionVersion()
```

##### Examples
```java
String extensionVersion = EdgeBridge.extensionVersion();
```

#### Kotlin

##### Examples
```kotlin
val extensionVersion = EdgeBridge.extensionVersion()
```
